# Portfolio Admin Panel

To'liq funksional admin panel portfolio website uchun.

## 🎯 Xususiyatlar

### ✅ Profile Management
- Profil ma'lumotlarini ko'rish va tahrirlash
- Ism, lavozim, bio, kontakt ma'lumotlari
- Social media linklar

### ✅ Skills Management
- Skills ro'yxatini ko'rish
- Yangi skill qo'shish
- Skillni tahrirlash
- Skillni o'chirish
- Skill level (1-100%)
- Kategoriyalar (Frontend, Backend, Database, Tools)

### ✅ Projects Management
- Loyihalar ro'yxati
- Yangi loyiha qo'shish
- Loyihani tahrirlash
- Loyihani o'chirish
- Featured loyihalar
- Technologies ro'yxati
- GitHub va Live URL linklar

### ✅ Experience Management
- Ish tajribasi ro'yxati
- Yangi tajriba qo'shish
- Tajribani tahrirlash
- Tajribani o'chirish
- Current position belgilash
- Sana oralig'i

### ✅ Testimonials Management
- Sharhlar ro'yxati
- Yangi sharh qo'shish
- Sharhni tahrirlash
- Sharhni o'chirish
- Rating (1-5 yulduz)

### ✅ Blog Management
- Blog postlar ro'yxati
- Yangi post yozish
- Postni tahrirlash
- Postni o'chirish
- Published/Draft status
- Tags
- Views counter

### ✅ Contact Messages
- Xabarlar ro'yxati
- Xabarni o'qish
- Xabarni o'chirish
- Read/Unread status

## 🚀 Ishga Tushirish

```bash
cd admin-panel
npm install
npm run dev
```

Admin Panel: http://localhost:5174
